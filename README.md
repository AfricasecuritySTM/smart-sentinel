# Smart Sentinel

### Real-Time Inventory Awareness Systems — Where Embedded Systems Meet Inventory Intelligence.

Smart Sentinel is an embedded systems and intelligent infrastructure company focused on building real-time inventory intelligence solutions using sensor systems, embedded computing, and IoT technologies.

Our current MVP, **Stock-GuardPro**, is an intelligent inventory monitoring system designed to connect physical stock movement to real-time digital awareness through load-cell sensing, embedded intelligence, and cloud-connected monitoring infrastructure.
